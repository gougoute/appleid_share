# Apple ID 共享 · 静态展示页

一个轻量、开箱即用的 **Apple ID 共享账号展示网站**。用户打开页面即可随机获取美区、港区等共享账号，支持一键复制账号与密码，界面采用 Apple 风格设计，在手机与电脑上都能舒适浏览。

**在线体验：** [https://appleid.gougouti.com/](https://appleid.gougouti.com/)

---

## 项目做什么

许多用户需要临时使用美区 / 港区 Apple ID 在 **App Store** 下载应用，本项目把「获取账号 → 展示 → 复制」做成单页体验：打开网站、点一下换号，就能拿到当前可用的共享账号信息。

## 主要特性

- **随机换号** — 点击「换一条账号」从聚合 API 拉取新的共享账号
- **一键复制** — 账号、密码字段支持复制到剪贴板，并带复制成功提示
- **安全提示** — 页面醒目说明：仅可在 App Store 登录，禁止登录 iCloud
- **响应式布局** — 针对手机竖屏优化，同时兼容桌面浏览器
- **SEO 友好** — 内置使用说明、常见问题与基础 meta / Open Graph 信息

---

## 使用须知

共享账号仅供在 **App Store** 登录下载应用使用。

- 请勿登录 **iCloud**，避免锁机或隐私风险
- 下载完成后请尽快退出共享账号，切回自己的 Apple ID
- 若提示需验证或无法登录，请刷新页面重新获取
